# UXAgent Paper

Reference paper:

**UXAgent: A System for Simulating Usability Testing of Web Design with LLM Agents**  
arXiv: `2504.09407`  
URL: https://arxiv.org/abs/2504.09407

Related project repository found during research:

- https://github.com/neuhai/uxagent

The paper describes a system for LLM-agent-based usability testing of web design. The relevant pattern for this repository is:

1. Generate or configure simulated user personas.
2. Give each persona a task.
3. Let an LLM-based browser agent interact with a website.
4. Capture actions, reasoning, screenshots, video replay, and task outcomes.
5. Review the resulting interaction evidence for usability findings.

This repository is not a fork of that implementation. It is a separate, reusable project concept inspired by the same problem area.
